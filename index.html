Websites as Zines

What is a website?

A website is a collection of files on your computer. There is usually one named "index.html" that holds all the information for the site. It is written in HTML. There are many great text-based sites out there that are written mostly using HTML, like this one.

If you're interested in hanging how your website looks, using colors and fonts, or nice layouts and grids, you're probably looking for something called CSS. CSS is a language used to apply styling to a site.

Lastly, if your site is going to collect user input, use data, or offer interaction (for example, with a form or buttons) you're probably going to need some JS (Javascript).

The internet is broken, but there are still some nice corners left.
